◆ Debug_SetLevel()

void Debug_SetLevel ( int  level)

Imposta il livello di severità utilizzato per la stampa delle informazioni di debug. Le informazioni sono stampate se il livello di severità è minore o uguale al livello impostato con il presente metodo. Ad es., se si imposta:

debug_setlevel(DBG_INFO);
Debug(DBG_INFO, "Info");
Debug(DBG_DEBUG, "Debug");

verrà stampata solo la stringa "Info".

Parametri
levelLivello di severità prevista per i messaggi di debug.
