7  The bits GK_ALT, GK_SHIFT and GK_CTRL indicates if these
8  modifier keys where pressed at the time the basic key where
9  pressed.
10  The modifier code is or'ed with the normal ASCII key or special
11  key.
12 
13  <Alt><F1> = GK_F1|GK_ALT
14  <Ctrl><a> = 'a'|GK_CTRL
15 
16  Note that the shift state is included with the shifted ASCII
17  value:
18  Lower case 'a'
19  <a> = 'a' or = 'a' & GK_KEY_MASK
20  Upper case 'A'
21  <Shift><a> = 'A'|GK_SHIFT or = 'A' & GK_KEY_MASK
